Configuration README file for Mini POS

The config.csv file is as follows with default values:

Row 00: [ID]                    [Value]
Row 01: [Language]              [English]
Row 02: [Currency Symbol]       [$]
Row 03: [Currency Decimals]     [2]
Row 04: [Toggle Thousands Sep]  [1]
Row 05: [Toggle Sales Tax]      [0]
Row 06: [Sales Tax]             [0.0]
